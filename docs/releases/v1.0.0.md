# Codex Config Switcher 1.0.0

首个完整里程碑版本，重点把项目从“可用原型”推进到“可分发工具”：

- macOS 主窗口、菜单栏、设置页和编辑区完成两轮视觉重构，操作层级更清晰。
- 预设管理补齐环境标签、收藏/最近使用、筛选排序、差异预览、操作历史和高风险确认。
- 本地模板库、模板工作台、导入导出和连接测试形成更完整的日常工作流。
- 新增跨平台 `CodexConfigSwitcherCLI`，并启动 Windows WinUI 3 GUI 桥接骨架。
- 新增 `dmg` 打包流程，方便直接通过 GitHub Release 分发 macOS 安装包。

## 安装包

- 文件：`Codex-Config-Switcher-1.0.0.dmg`
- 平台：macOS 13+
- SHA256：`c90c12fe69e81a42bce3060cd2bfc405ab630c320679553f3acb39eedb5c7c34`

## 安装方式

1. 下载 `Codex-Config-Switcher-1.0.0.dmg`。
2. 打开镜像后，将 `Codex Config Switcher.app` 拖到 `Applications`。
3. 首次打开如果遇到系统安全提示，请到“系统设置 -> 隐私与安全性”中手动放行。

## 本次重点更新

- 支持基础模式/高级模式编辑、应用前差异预览、键盘快捷键和 live/draft 上下文提示。
- 支持模板保存、模板工作台、从模板快速新建，以及更完整的导入/导出/校验流程。
- 支持目标应用状态检查、重启控制和跨平台默认路径抽象。
- 提供 `status --json` 与 `target status --json`，为 Windows GUI 继续扩展保留桥接面。

- Windows WinUI 3 工程已入仓，但本次 Release 仍以 macOS 安装包为主。
