# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-04-17

### Added

- 支持环境标签、收藏与最近使用分组、筛选排序、操作历史和应用前差异预览，提升多预设管理效率。
- 支持基础模式/高级模式分层编辑、连接测试、应用内快捷键和 live/draft 上下文提示，减少误操作成本。
- 支持本地模板库、模板工作台、模板快速新建，以及更完整的预设导入导出与校验流程。
- 新增跨平台 `CodexConfigSwitcherCLI`，支持状态查询、预设应用、目标应用状态/重启和模板相关操作。
- 新增 Windows 默认路径抽象、WinUI 3 工程骨架和 CLI JSON 桥，为后续 Windows GUI 打下基础。
- 新增 `.dmg` 打包脚本，支持生成可直接附加到 GitHub Release 的 macOS 安装镜像。

### Changed

- 重构 macOS 主窗口、菜单栏、设置页和编辑区的视觉层级，统一卡片、徽章、按钮和 hover 反馈样式。
- 调整主窗口首屏信息密度与快捷动作分级，让“当前生效状态 -> 高频动作 -> 编辑区”的路径更清晰。

## [0.1.0] - 2026-04-09

### Added

- 初始化 `Codex Config Switcher` macOS 原生应用工程。
- 支持读取和编辑 `~/.codex/config.toml` 与 `~/.codex/auth.json`。
- 支持保存、编辑、应用多个配置预设。
- 支持从当前 live 配置导入预设。
- 支持写入前自动备份原始配置文件。
- 支持应用配置后提示是否自动重启目标软件，默认联动 `Codex`。
- 提供菜单栏入口、主窗口编辑界面、图标资源与 `.app` 打包脚本。
- 提供核心配置文件服务与预设持久化测试。
